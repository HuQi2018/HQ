##git学习记录
###git基本操作：
1.仓库初始化：git init
2.添加文件内容至索引：git add .（.代表当前目录所有文件）
3.切换分支或恢复工作区文件：git checkout
4.记录变更到仓库：git commit -m "name"
5.更新远程引用和相关对象：git push origin master
6.获取并整合另外的仓库或一个本地分支：git pull
7.克隆一个仓库到本地目录：git clone URL
8.现实工作区状态：git status
